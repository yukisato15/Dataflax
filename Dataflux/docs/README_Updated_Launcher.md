### 機能テスト手順

1. **全カテゴリ確認**
   - ランチャー起動後、4つのセクションを確認
   - 実装済み（緑/青）と計画中（グレー）の識別

2. **実装済み機能テスト**
   - 🔍 フォルダ解析: PySide6版解析ツール起動
   - 🎵 Audio: Tkinter版音声ツール起動

3. **プレースホルダー機能**
   - 🎥 Video等の計画中ツールをクリック
   - 詳細ウィンドウで3つのタブを確認
   - 機能説明・スケジュール・技術情報

4. **テーマ切替**
   - 右上のテーマ選択で Pro ↔ Light 切替
   - 全要素の色変更を確認

5. **従来ツール連携**
   - ユーティリティセクションから従来版起動
   - Tkinter版との併用確認
