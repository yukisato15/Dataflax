# Workspace Layout

この `databox` は整理済みで、Dataflux 本体は以下にあります。

- `Dataflux/`

## 起動方法

旧手順との互換のため、ルートにもラッパーを置いています。

```bash
cd /Users/yukisato/PythonProjects.pdf/databox
python3 main.py
```

または直接:

```bash
cd /Users/yukisato/PythonProjects.pdf/databox/Dataflux
python3 main.py
```

## ビルド

```bash
cd /Users/yukisato/PythonProjects.pdf/databox
python3 build_dataflux.py
```

上記は `Dataflux/build_dataflux.py` を呼び出します。
